dialogic-vxml-netann
====================

Overview: A JAVA based repository using the PowerMedia XMS RESTful API which provides application developers using RESTful API over http transport to control media and call control resources of PowerMedia XMS.

The PowerMedia VoiceXML Reference Guide contains an alphabetical reference of supported VoiceXML elements and provides information about application properties, SSML support, session variables, and application variables.

The PowerMedia XMS Basic Network Media Services with SIP User's Guide provides detailed information about configuring Basic Network Media Services with SIP, focusing on Network Announcement (NETANN).
